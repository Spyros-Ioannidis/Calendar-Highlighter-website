# Calendar-Highlighter-website
Είναι μία απλή εφαρμογή HTML/CSS/JAVASCRIPT με σκοπό να επισημάνει ημερομηνίες που θα δοθούν από τον χρήστη σε ημερολόγιο

[App demo](https://spyros-ioannidis.github.io/Calendar-Highlighter-website/)
## Πληροφορίες
Η εισαγωγή ημερομηνιών γίνεται μέσω αρχείου txt.

Υπάρχουν info icons μέσα στην εφαρμογή για επιπλέον πληροφορίες χρήσης της.

### Μέσα στην εφαρμογή υπάρχουν επιλογές για:
- Επιλογή χρονίας
- Τρόπου εμφάνισης ημερολόγιου και στήλες
- Εμφάνιση prefix string  μπροστά στις επισημασμένες ημερομηνίες στα tooltips\
(Εμφανίζεται και ένας αριθμός που είναι η σειρά της ημερομηνίας στο αρχείο txt)
- Επιλογή αρχείου και κουμπί δοκιμής

- Console logs των χρονομετρήσεων των functions της εφαρμογής
